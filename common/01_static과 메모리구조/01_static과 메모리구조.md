## 정리
- static로 지정된 데이터는 컴파일 단계에서 저장되는 데이터로, 프로그램 시작과 동시에 할당된다.
  그러하다보니, 메모리 할당을 맨 처음 한번만 하게되며,
  -> 메모리 공간 절약
  -> 코드 전반에 걸쳐, 해당 메모리공간 한곳만 바라보다보니, 데이터 공유
  이러한 부수 효과를 얻을 수 있다

  즉, static 데이터는 컴파일시, 한번만 할당되는 데이터이다! 라는 사실이 핵심이며, 데이터 공유 자체가 static을 의미하는 것이 아니다.
